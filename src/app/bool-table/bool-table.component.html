How to enter a valid formula

Use the letters A-Z and a-z for the variables in your formula. Please note that uppercase and lowercase variables (like "A" and "a") are considered as different variables.

You can enter Operators by using the Buttons next to the input panel or type these codes for the different operators:

not: ¬
nand: ⊼
and: ∧
nor: ⊽
or: ∨
xor: ⊻
impr: →
impl: ←
equ: ↔

You can use round brackets to influence the priority of operations
Without brackets the priority of operators is the following (from highest to lowest priority):
¬, ⊼, ∧, ⊽, ∨, ⊻, →, ←, ↔
